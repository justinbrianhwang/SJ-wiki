## Topics

### Foundations

- **[Programming Languages](/cs/programming/intro)** — C, C++, Rust, Java, Python, R
- **[Algorithms](/cs/algorithms/intro)** — sorting, searching, DP, graph algorithms
- **[Data Structures](/cs/data-structures/intro)** — arrays, trees, hash tables, graphs
- **[Theory of Computation](/cs/theory/intro)** — automata, complexity, computability

### Systems

- **[Operating Systems](/cs/operating-systems/intro)** — processes, scheduling, memory, files
- **[Databases](/cs/databases/intro)** — relational model, transactions, indexing
- **[Computer Architecture](/cs/computer-architecture/intro)** — ISA, pipelines, caches, parallelism
- **[Embedded & Microprocessors](/cs/embedded/intro)** — 8085, 8051, microcontrollers, peripherals
- **[Control Engineering](/cs/control-engineering/intro)** — Laplace, root locus, Bode/Nyquist, state-space, digital control

### Machine Learning

- **[Data Mining](/cs/data-mining/intro)** — clustering, classification, association rules
- **[Machine Learning](/cs/machine-learning/intro)** — classical statistical learning, decision trees, Bayesian methods
- **[Deep Learning](/cs/deep-learning/intro)** — CNNs, RNNs, Transformers, modern architectures
- **[Reinforcement Learning](/cs/reinforcement-learning/intro)** — MDPs, Q-learning, policy gradients, deep RL
- **[Adversarial Attacks](/cs/adversarial-attacks/intro)** — robustness, white/black-box attacks, defenses, certified bounds

### Applied

- **[Natural Language Processing](/cs/nlp/intro)** — language models, parsing, semantics
- **[Cryptography](/cs/cryptography/intro)** — symmetric/public-key, security proofs
- **[Software Engineering](/cs/software-engineering/intro)** — design, testing, process
- **[Autonomous Driving](/cs/autonomous-driving/intro)** — perception, prediction, planning, control, safety

## Languages used in code samples

- **Python** — algorithmic clarity, ML/DL/NLP, data mining
- **C / C++** — systems, data structures, embedded
- **Rust** — safe systems programming
- **Java** — OOP, concurrency
- **R** — statistics and visualization
- **MATLAB** — control engineering, signal processing
