System Properties

A system maps an input signal to an output signal. It can represent an electrical network, a mechanical suspension, a digital filter, a communication channel, or an abstract operation such as differentiation. Before solving a system, the first task is to classify it. Properties such as linearity, time invariance, causality, memory, stability, and invertibility determine which tools are legal and which conclusions can be trusted.

The most important later theory, linear time-invariant analysis, is built from two of these properties. Linearity gives superposition. Time invariance says the system behaves the same way regardless of the absolute time origin. Together they imply that the response to any signal can be assembled from shifted impulse responses, producing convolution.

Definitions​

A continuous-time system TTT maps an input x(t)x(t)x(t) to an output

y(t)=T{x(t)}.y(t)=T\{x(t)\}.y(t)=T{x(t)}.

A discrete-time system maps x[n]x[n]x[n] to

y[n]=T{x[n]}.y[n]=T\{x[n]\}.y[n]=T{x[n]}.

A system is memoryless if the output at a time depends only on the input value at that same time. In continuous time, a memoryless system has the form

y(t)=f(x(t))y(t)=f(x(t))y(t)=f(x(t))

for some function fff. In discrete time,

y[n]=f(x[n]).y[n]=f(x[n]).y[n]=f(x[n]).

A system has memory if the output depends on past, future, or accumulated values of the input. Examples include

y(t)=x(t−2),y(t)=∫−∞tx(τ) dτ,y[n]=x[n]+x[n−1].y(t)=x(t-2), \qquad y(t)=\int_{-\infty}^{t}x(\tau)\,d\tau, \qquad y[n]=x[n]+x[n-1].y(t)=x(t−2),y(t)=∫−∞t​x(τ)dτ,y[n]=x[n]+x[n−1].

A system is linear if it satisfies superposition. For any inputs x1x_1x1​ and x2x_2x2​, and any constants aaa and bbb,

T{ax1+bx2}=aT{x1}+bT{x2}.T\{a x_1+b x_2\}=aT\{x_1\}+bT\{x_2\}.T{ax1​+bx2​}=aT{x1​}+bT{x2​}.

Equivalently, linearity is additivity plus homogeneity.

A system is time-invariant if shifting the input shifts the output by the same amount. In continuous time, if

T{x(t)}=y(t),T\{x(t)\}=y(t),T{x(t)}=y(t),

then for every real t0t_0t0​,

T{x(t−t0)}=y(t−t0).T\{x(t-t_0)\}=y(t-t_0).T{x(t−t0​)}=y(t−t0​).

In discrete time, if T{x[n]}=y[n]T\{x[n]\}=y[n]T{x[n]}=y[n], then for every integer n0n_0n0​,

T{x[n−n0]}=y[n−n0].T\{x[n-n_0]\}=y[n-n_0].T{x[n−n0​]}=y[n−n0​].

A system is causal if the output at the present does not depend on future input values. For continuous time, y(t0)y(t_0)y(t0​) may depend on x(t)x(t)x(t) only for t≤t0t\le t_0t≤t0​. For discrete time, y[n0]y[n_0]y[n0​] may depend on x[n]x[n]x[n] only for n≤n0n\le n_0n≤n0​.

A system is bounded-input bounded-output stable, or BIBO stable, if every bounded input produces a bounded output. Formally, if

∣x(t)∣≤Bx<∞|x(t)|\le B_x<\infty∣x(t)∣≤Bx​<∞

for all ttt, then there must exist a finite ByB_yBy​ such that

∣y(t)∣≤By|y(t)|\le B_y∣y(t)∣≤By​

for all ttt. The same definition applies to sequences.

A system is invertible if distinct inputs produce distinct outputs, so that another system can recover the input from the output. If T−1T^{-1}T−1 exists, then

T−1{T{x}}=x.T^{-1}\{T\{x\}\}=x.T−1{T{x}}=x.

Key results​

To prove linearity, test the combined input directly. Compute the output produced by ax1+bx2a x_1+b x_2ax1​+bx2​, and compare it with ay1+by2a y_1+b y_2ay1​+by2​. A single counterexample is enough to disprove linearity. Systems that add constants, multiply signals by themselves, take magnitudes, or contain fixed thresholds are usually nonlinear.

To test time invariance, use two paths:

  1. Shift the input first, then apply the system.
  2. Apply the system first, then shift the output.

If the two expressions match for every input and every shift, the system is time-invariant. If the system contains explicit time, such as tx(t)t x(t)tx(t) or nx[n]n x[n]nx[n], it is often time-varying. However, explicit time is a warning sign, not a proof by itself; the two-path test is definitive.

For memorylessness, a delay system such as y[n]=x[n−1]y[n]=x[n-1]y[n]=x[n−1] has memory, even though it is simple. A memoryless nonlinear system such as

y(t)=x2(t)y(t)=x^2(t)y(t)=x2(t)

has no memory but is not linear. Properties are independent; one should not infer one from another.

Causality is about information ordering, not algebraic simplicity. The system

y[n]=x[n]+x[n−1]y[n]=x[n]+x[n-1]y[n]=x[n]+x[n−1]

is causal because it uses present and past inputs. The system

y[n]=x[n+1]y[n]=x[n+1]y[n]=x[n+1]

is noncausal because it requires a future input sample.

For LTI systems, later pages give sharper tests: causality is equivalent to h(t)=0h(t)=0h(t)=0 for t<0t\lt 0t<0 or h[n]=0h[n]=0h[n]=0 for n<0n\lt 0n<0, and BIBO stability is equivalent to absolute integrability or absolute summability of the impulse response. Those tests depend on LTI structure and should not be used for arbitrary systems.

When several properties are requested, test them independently and record the reason for each conclusion. A system can be linear and noncausal, nonlinear and memoryless, stable and noninvertible, or causal and unstable. Many exam errors come from treating the properties as a chain of implications. The only major implication in this early list is that memoryless systems are automatically causal, because a present output depending only on the present input cannot depend on future input values.

For systems described by formulas, it is helpful to classify the location of input samples or values first. Terms like x(t−3)x(t-3)x(t−3) or x[n−1]x[n-1]x[n−1] point to the past. Terms like x(t+3)x(t+3)x(t+3) or x[n+1]x[n+1]x[n+1] point to the future. Terms like ∫−∞tx(τ)dτ\int_{-\infty}^{t}x(\tau)d\tau∫−∞t​x(τ)dτ use accumulated past information, while ∫t∞x(τ)dτ\int_{t}^{\infty}x(\tau)d\tau∫t∞​x(τ)dτ uses future information. This quick scan makes the causality and memory tests less error-prone before moving to linearity and time invariance.

Worked example 1: classifying a continuous-time system​

Problem: Classify

y(t)=tx(t−1)y(t)=t x(t-1)y(t)=tx(t−1)

with respect to memory, linearity, time invariance, and causality.

Method:

  1. Memory: The output at time ttt uses x(t−1)x(t-1)x(t−1), not only x(t)x(t)x(t). Therefore the system has memory.

  2. Linearity: Let inputs x1x_1x1​ and x2x_2x2​ produce

y1(t)=tx1(t−1),y2(t)=tx2(t−1).y_1(t)=t x_1(t-1), \qquad y_2(t)=t x_2(t-1).y1​(t)=tx1​(t−1),y2​(t)=tx2​(t−1).

For input ax1+bx2a x_1+b x_2ax1​+bx2​,

T{ax1+bx2}=t(ax1(t−1)+bx2(t−1)).T\{a x_1+b x_2\} =t\left(a x_1(t-1)+b x_2(t-1)\right).T{ax1​+bx2​}=t(ax1​(t−1)+bx2​(t−1)).

Distribute:

t(ax1(t−1)+bx2(t−1))=atx1(t−1)+btx2(t−1)=ay1(t)+by2(t).t\left(a x_1(t-1)+b x_2(t-1)\right) =a t x_1(t-1)+b t x_2(t-1) =a y_1(t)+b y_2(t).t(ax1​(t−1)+bx2​(t−1))=atx1​(t−1)+btx2​(t−1)=ay1​(t)+by2​(t).

So the system is linear.

  1. Time invariance: First shift the input by t0t_0t0​:
xs(t)=x(t−t0).x_s(t)=x(t-t_0).xs​(t)=x(t−t0​).

Apply the system:

T{xs}=txs(t−1)=tx(t−1−t0).T\{x_s\}=t x_s(t-1)=t x(t-1-t_0).T{xs​}=txs​(t−1)=tx(t−1−t0​).

Now apply the system first and shift the output:

y(t−t0)=(t−t0)x(t−t0−1).y(t-t_0)=(t-t_0)x(t-t_0-1).y(t−t0​)=(t−t0​)x(t−t0​−1).

The two expressions are

tx(t−1−t0)t x(t-1-t_0)tx(t−1−t0​)

and

(t−t0)x(t−1−t0).(t-t_0)x(t-1-t_0).(t−t0​)x(t−1−t0​).

They are not equal for arbitrary t0t_0t0​ and xxx. Therefore the system is time-varying.

  1. Causality: At time ttt, the output uses x(t−1)x(t-1)x(t−1), a past value. Therefore the system is causal.

Checked answer: The system has memory, is linear, is time-varying, and is causal.

Worked example 2: testing BIBO stability and invertibility​

Problem: Consider the discrete-time system

y[n]=x2[n].y[n]=x^2[n].y[n]=x2[n].

Determine whether it is memoryless, linear, time-invariant, BIBO stable, and invertible over real-valued input sequences.

Method:

  1. Memoryless: y[n]y[n]y[n] depends only on x[n]x[n]x[n], so it is memoryless.

  2. Linearity: Test homogeneity with input x[n]x[n]x[n] and scalar aaa:

T{ax[n]}=(ax[n])2=a2x2[n].T\{a x[n]\}=(a x[n])^2=a^2x^2[n].T{ax[n]}=(ax[n])2=a2x2[n].

But

aT{x[n]}=ax2[n].aT\{x[n]\}=a x^2[n].aT{x[n]}=ax2[n].

These are not equal for general aaa. For example, a=2a=2a=2 gives 4x2[n]4x^2[n]4x2[n] versus 2x2[n]2x^2[n]2x2[n]. The system is nonlinear.

  1. Time invariance: Shift the input:
xs[n]=x[n−n0].x_s[n]=x[n-n_0].xs​[n]=x[n−n0​].

Then

T{xs}=x2[n−n0].T\{x_s\}=x^2[n-n_0].T{xs​}=x2[n−n0​].

The original output shifted is

y[n−n0]=x2[n−n0].y[n-n_0]=x^2[n-n_0].y[n−n0​]=x2[n−n0​].

They match, so the system is time-invariant.

  1. BIBO stability: If ∣x[n]∣≤Bx\vert x[n]\vert \le B_x∣x[n]∣≤Bx​, then
∣y[n]∣=∣x2[n]∣=∣x[n]∣2≤Bx2.|y[n]|=|x^2[n]|=|x[n]|^2\le B_x^2.∣y[n]∣=∣x2[n]∣=∣x[n]∣2≤Bx2​.

So the system is BIBO stable.

  1. Invertibility: Real inputs x[n]x[n]x[n] and −x[n]-x[n]−x[n] produce the same output:
x2[n]=(−x[n])2.x^2[n]=(-x[n])^2.x2[n]=(−x[n])2.

Unless the domain is restricted to nonnegative signals, the input cannot be recovered uniquely.

Checked answer: The system is memoryless, nonlinear, time-invariant, BIBO stable, and not invertible over all real sequences.

Code​

import numpy as np

def system_square(x):
    return x**2

n = np.arange(-4, 5)
x = np.array([-2, -1, 0, 1, 2, 1, 0, -1, -2], dtype=float)
shift = 2

left = system_square(np.roll(x, shift))
right = np.roll(system_square(x), shift)

print("time-invariance check:", np.allclose(left, right))
print("x and -x produce same output:", np.allclose(system_square(x), system_square(-x)))
print("bounded input max:", np.max(np.abs(x)))
print("bounded output max:", np.max(np.abs(system_square(x))))

Common pitfalls​

  • Treating nonlinear as equivalent to time-varying. These are separate properties.
  • Calling a delayed system memoryless. Any dependence on x(t−t0)x(t-t_0)x(t−t0​) with t0≠0t_0\ne 0t0​=0 is memory.
  • Declaring a system time-varying only because the expression looks complicated. Use the two-path shift test.
  • Using LTI impulse-response tests for systems that have not been proven LTI.
  • Forgetting that invertibility depends on the allowed input set. Squaring is not invertible on all real signals, but it is invertible on nonnegative signals.
