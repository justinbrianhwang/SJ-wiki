Sets and Set Operations

Sets are the basic containers of discrete mathematics. Relations are sets of ordered pairs, graphs are sets of vertices and edges, languages are sets of strings, and events in probability are sets of outcomes. The discipline around sets is simple but strict: order does not matter, repetition does not matter, and membership is the central question.

Set notation gives a compact way to describe large or infinite collections. Instead of listing every object, we can define a set by a property, combine sets with operations, and prove two descriptions match by checking membership element by element. This makes set theory the bridge between logic and the rest of the subject.

Definitions​

A set is an unordered collection of distinct elements. Write a∈Aa\in Aa∈A when aaa is an element of AAA, and a∉Aa\notin Aa∈/A otherwise. The same set can be described by roster notation, such as {1,2,3}\{1,2,3\}{1,2,3}, or set-builder notation, such as {x∈Z:1≤x≤3}\{x\in\mathbb{Z}:1\le x\le3\}{x∈Z:1≤x≤3}.

Two sets are equal exactly when they have the same elements:

A=Bif and only if∀x(x∈A↔x∈B).A=B \quad\text{if and only if}\quad \forall x(x\in A \leftrightarrow x\in B).A=Bif and only if∀x(x∈A↔x∈B).

AAA is a subset of BBB, written A⊆BA\subseteq BA⊆B, if every element of AAA is also in BBB:

A⊆Bif and only if∀x(x∈A→x∈B).A\subseteq B \quad\text{if and only if}\quad \forall x(x\in A \to x\in B).A⊆Bif and only if∀x(x∈A→x∈B).

If A⊆BA\subseteq BA⊆B and A≠BA\ne BA=B, then AAA is a proper subset of BBB. The empty set ∅\emptyset∅ has no elements and is a subset of every set. The power set P(A)\mathcal{P}(A)P(A) is the set of all subsets of AAA.

For sets within a universal set UUU:

  • Union: A∪B={x:x∈A∨x∈B}A\cup B=\{x:x\in A\lor x\in B\}A∪B={x:x∈A∨x∈B}.
  • Intersection: A∩B={x:x∈A∧x∈B}A\cap B=\{x:x\in A\land x\in B\}A∩B={x:x∈A∧x∈B}.
  • Difference: A−B={x:x∈A∧x∉B}A-B=\{x:x\in A\land x\notin B\}A−B={x:x∈A∧x∈/B}.
  • Complement: A‾=U−A\overline{A}=U-AA=U−A.
  • Symmetric difference: A⊕B=(A−B)∪(B−A)A\oplus B=(A-B)\cup(B-A)A⊕B=(A−B)∪(B−A).
  • Cartesian product: A×B={(a,b):a∈A∧b∈B}A\times B=\{(a,b):a\in A\land b\in B\}A×B={(a,b):a∈A∧b∈B}.

Sets can contain other sets. For example, if A={1,2}A=\{1,2\}A={1,2}, then P(A)={∅,{1},{2},{1,2}}\mathcal{P}(A)=\{\emptyset,\{1\},\{2\},\{1,2\}\}P(A)={∅,{1},{2},{1,2}}. Notice that 1∈A1\in A1∈A, but {1}⊆A\{1\}\subseteq A{1}⊆A and {1}∈P(A)\{1\}\in\mathcal{P}(A){1}∈P(A).

Key results​

Set identities mirror logical equivalences. De Morgan's laws for sets are:

A∪B‾=A‾∩B‾,A∩B‾=A‾∪B‾.\overline{A\cup B}=\overline{A}\cap\overline{B}, \qquad \overline{A\cap B}=\overline{A}\cup\overline{B}.A∪B=A∩B,A∩B=A∪B.

Proof of the first identity by element chasing: let xxx be arbitrary. Then

x∈A∪B‾↔x∉A∪B↔¬(x∈A∨x∈B)↔x∉A∧x∉B↔x∈A‾∩B‾.\begin{aligned} x\in\overline{A\cup B} &\leftrightarrow x\notin A\cup B\\ &\leftrightarrow \neg(x\in A\lor x\in B)\\ &\leftrightarrow x\notin A\land x\notin B\\ &\leftrightarrow x\in\overline{A}\cap\overline{B}. \end{aligned}x∈A∪B​↔x∈/A∪B↔¬(x∈A∨x∈B)↔x∈/A∧x∈/B↔x∈A∩B.​

Since the equivalence holds for every xxx, the sets are equal.

Other useful identities include:

A∪(B∩C)=(A∪B)∩(A∪C),A∩(B∪C)=(A∩B)∪(A∩C),A∪∅=A,A∩U=A,A∪A‾=U,A∩A‾=∅.\begin{aligned} A\cup(B\cap C)&=(A\cup B)\cap(A\cup C),\\ A\cap(B\cup C)&=(A\cap B)\cup(A\cap C),\\ A\cup\emptyset&=A,\\ A\cap U&=A,\\ A\cup\overline{A}&=U,\\ A\cap\overline{A}&=\emptyset. \end{aligned}A∪(B∩C)A∩(B∪C)A∪∅A∩UA∪AA∩A​=(A∪B)∩(A∪C),=(A∩B)∪(A∩C),=A,=A,=U,=∅.​

If AAA is finite with nnn elements, then ∣P(A)∣=2n\vert \mathcal{P}(A)\vert =2^n∣P(A)∣=2n. Each element of AAA has two independent choices when forming a subset: include it or exclude it. By the product rule, this gives 2n2^n2n subsets.

For finite sets, inclusion-exclusion begins with

∣A∪B∣=∣A∣+∣B∣−∣A∩B∣.|A\cup B|=|A|+|B|-|A\cap B|.∣A∪B∣=∣A∣+∣B∣−∣A∩B∣.

The subtraction corrects the double count of elements lying in both sets. This same idea grows into the larger inclusion-exclusion formulas used in counting.

Worked example 1: Compute operations and check inclusion-exclusion​

Problem. Let

U={1,2,3,4,5,6,7,8},A={1,2,3,5,8},B={2,4,5,6}.U=\{1,2,3,4,5,6,7,8\},\quad A=\{1,2,3,5,8\},\quad B=\{2,4,5,6\}.U={1,2,3,4,5,6,7,8},A={1,2,3,5,8},B={2,4,5,6}.

Find A∪BA\cup BA∪B, A∩BA\cap BA∩B, A−BA-BA−B, A‾\overline{A}A, and verify inclusion-exclusion.

Method.

  1. The union contains elements appearing in either set:
A∪B={1,2,3,4,5,6,8}.A\cup B=\{1,2,3,4,5,6,8\}.A∪B={1,2,3,4,5,6,8}.
  1. The intersection contains elements appearing in both:
A∩B={2,5}.A\cap B=\{2,5\}.A∩B={2,5}.
  1. The difference A−BA-BA−B keeps elements of AAA that are not in BBB:
A−B={1,3,8}.A-B=\{1,3,8\}.A−B={1,3,8}.
  1. The complement of AAA is computed relative to UUU:
A‾=U−A={4,6,7}.\overline{A}=U-A=\{4,6,7\}.A=U−A={4,6,7}.
  1. Check inclusion-exclusion:
∣A∣+∣B∣−∣A∩B∣=5+4−2=7=∣A∪B∣.\begin{aligned} |A|+|B|-|A\cap B| &=5+4-2\\ &=7\\ &=|A\cup B|. \end{aligned}∣A∣+∣B∣−∣A∩B∣​=5+4−2=7=∣A∪B∣.​

Checked answer. The operations are correct, and the count agrees. Element 777 is in neither AAA nor BBB, which is why it does not appear in the union even though it is in the universal set.

Worked example 2: Prove a set identity by element chasing​

Problem. Prove

A−(B∪C)=(A−B)∩(A−C).A-(B\cup C)=(A-B)\cap(A-C).A−(B∪C)=(A−B)∩(A−C).

Method.

  1. Let xxx be an arbitrary element.
  2. Translate membership in the left side:
x∈A−(B∪C)↔x∈A∧x∉B∪C↔x∈A∧¬(x∈B∨x∈C)↔x∈A∧x∉B∧x∉C.\begin{aligned} x\in A-(B\cup C) &\leftrightarrow x\in A\land x\notin B\cup C\\ &\leftrightarrow x\in A\land \neg(x\in B\lor x\in C)\\ &\leftrightarrow x\in A\land x\notin B\land x\notin C. \end{aligned}x∈A−(B∪C)​↔x∈A∧x∈/B∪C↔x∈A∧¬(x∈B∨x∈C)↔x∈A∧x∈/B∧x∈/C.​
  1. Translate membership in the right side:
x∈(A−B)∩(A−C)↔x∈A−B∧x∈A−C↔(x∈A∧x∉B)∧(x∈A∧x∉C)↔x∈A∧x∉B∧x∉C.\begin{aligned} x\in (A-B)\cap(A-C) &\leftrightarrow x\in A-B\land x\in A-C\\ &\leftrightarrow (x\in A\land x\notin B)\land(x\in A\land x\notin C)\\ &\leftrightarrow x\in A\land x\notin B\land x\notin C. \end{aligned}x∈(A−B)∩(A−C)​↔x∈A−B∧x∈A−C↔(x∈A∧x∈/B)∧(x∈A∧x∈/C)↔x∈A∧x∈/B∧x∈/C.​
  1. Both sides have the same membership condition for arbitrary xxx.

Checked answer. Since every element belongs to the left side exactly when it belongs to the right side, the sets are equal. This proof also shows why the formula is a set version of De Morgan's law.

Code​

from itertools import chain, combinations

def powerset(items):
    items = list(items)
    for r in range(len(items) + 1):
        for combo in combinations(items, r):
            yield set(combo)

U = set(range(1, 9))
A = {1, 2, 3, 5, 8}
B = {2, 4, 5, 6}

print("union", A | B)
print("intersection", A & B)
print("difference", A - B)
print("complement", U - A)

Python sets implement union, intersection, difference, and symmetric difference directly. The powerset function uses combinations of all possible subset sizes.

Common pitfalls​

  • Confusing ∈\in∈ and ⊆\subseteq⊆. An element belongs to a set; a set can be a subset of another set.
  • Forgetting the universal set when taking complements. A‾\overline{A}A depends on the surrounding UUU.
  • Treating repeated roster entries as distinct. {1,1,2}={1,2}\{1,1,2\}=\{1,2\}{1,1,2}={1,2}.
  • Assuming A−B=B−AA-B=B-AA−B=B−A. Set difference is not commutative.
  • Treating Cartesian products as unordered. Usually (a,b)≠(b,a)(a,b)\ne(b,a)(a,b)=(b,a).
  • Proving a set identity by drawing only one picture. Diagrams build intuition, but element chasing provides a proof.

Set-builder notation should always include a domain when ambiguity is possible. The expression {x:x2<4}\{x:x^2\lt 4\}{x:x2<4} means different things over integers, rationals, and real numbers. Writing {x∈Z:x2<4}\{x\in\mathbb{Z}:x^2\lt 4\}{x∈Z:x2<4} makes the intended set finite and equal to {−1,0,1}\{-1,0,1\}{−1,0,1}, while the real-domain version is the interval (−2,2)(-2,2)(−2,2).

To prove A=BA=BA=B, use the element method or prove two subset inclusions. The element method shows x∈A↔x∈Bx\in A\leftrightarrow x\in Bx∈A↔x∈B for arbitrary xxx. The subset method proves A⊆BA\subseteq BA⊆B and B⊆AB\subseteq AB⊆A separately. Both approaches are formal versions of "same elements," and both avoid relying on a diagram that may not capture all cases.

Power sets are a common source of type errors. If A={1,2}A=\{1,2\}A={1,2}, then {1}∈P(A)\{1\}\in\mathcal{P}(A){1}∈P(A) and {1}⊆A\{1\}\subseteq A{1}⊆A, but {1}∉A\{1\}\notin A{1}∈/A unless AAA itself contains the set {1}\{1\}{1} as an element. Keeping track of whether an object is an element or a set of elements prevents many mistakes in relations and functions.

Cartesian products are ordered and can change size quickly. If ∣A∣=m\vert A\vert =m∣A∣=m and ∣B∣=n\vert B\vert =n∣B∣=n, then ∣A×B∣=mn\vert A\times B\vert =mn∣A×B∣=mn. If A=BA=BA=B, the product A×AA\times AA×A contains ordered pairs, so (a,b)(a,b)(a,b) and (b,a)(b,a)(b,a) are different when a≠ba\ne ba=b. Relations on AAA are subsets of this ordered product, which is why there are 2∣A∣22^{\vert A\vert ^2}2∣A∣2 possible relations on a finite set AAA.

For complements and differences, always name the universal set. In probability, the complement of an event is relative to the sample space. In number theory, the complement of even integers might mean odd integers within Z\mathbb{Z}Z, but within N\mathbb{N}N it means positive odd integers. The notation alone does not determine the surrounding universe.

For finite examples, verify identities by testing membership rather than by comparing roster lists too early. Roster lists can hide duplicates or ordering changes. The element method forces the exact logical condition on both sides, which is why it scales from three-element examples to arbitrary sets.

When a problem involves several operations, add parentheses before simplifying. Set difference binds less predictably in students' minds than union and intersection, and A−(B−C)A-(B-C)A−(B−C) is not the same as (A−B)−C(A-B)-C(A−B)−C. Translating every operation into membership logic is the most reliable way to disambiguate the expression.

For finite sets, cardinality checks are useful but not complete proofs of equality unless one subset relation is already known. Two sets can have the same size and different elements. If you prove A⊆BA\subseteq BA⊆B and ∣A∣=∣B∣\vert A\vert =\vert B\vert ∣A∣=∣B∣ for finite sets, then equality follows; without the subset relation, equal size alone is not enough.
