Vector Functions and Motion

Vector functions describe curves and motion in the plane or in space. Instead of tracking one output, a vector function tracks several coordinates at once. The derivative of a vector function gives velocity, and the second derivative gives acceleration. This makes vector calculus a natural extension of parametric calculus.

Motion problems become clearer when position, velocity, acceleration, speed, tangent direction, and curvature are separated. The vector notation keeps direction and magnitude together while still allowing ordinary differentiation component by component.

Definitions​

A vector-valued function in space has form

r(t)=⟨x(t),y(t),z(t)⟩.\mathbf{r}(t)=\langle x(t),y(t),z(t)\rangle.r(t)=⟨x(t),y(t),z(t)⟩.

Its derivative is

r′(t)=⟨x′(t),y′(t),z′(t)⟩.\mathbf{r}'(t)=\langle x'(t),y'(t),z'(t)\rangle.r′(t)=⟨x′(t),y′(t),z′(t)⟩.

For motion, position is r(t)\mathbf{r}(t)r(t), velocity is

v(t)=r′(t),\mathbf{v}(t)=\mathbf{r}'(t),v(t)=r′(t),

and acceleration is

a(t)=v′(t)=r′′(t).\mathbf{a}(t)=\mathbf{v}'(t)=\mathbf{r}''(t).a(t)=v′(t)=r′′(t).

Speed is the magnitude of velocity:

∣v(t)∣.|\mathbf{v}(t)|.∣v(t)∣.

The unit tangent vector is

T(t)=r′(t)∣r′(t)∣\mathbf{T}(t)=\frac{\mathbf{r}'(t)}{|\mathbf{r}'(t)|}T(t)=∣r′(t)∣r′(t)​

when r′(t)≠0\mathbf{r}'(t)\ne \mathbf{0}r′(t)=0.

Arc length from t=at=at=a to t=bt=bt=b is

L=∫ab∣r′(t)∣ dt.L=\int_a^b |\mathbf{r}'(t)|\,dt.L=∫ab​∣r′(t)∣dt.

Curvature measures how quickly the tangent direction changes with respect to arc length. In space,

κ(t)=∣r′(t)×r′′(t)∣∣r′(t)∣3\kappa(t)=\frac{|\mathbf{r}'(t)\times\mathbf{r}''(t)|}{|\mathbf{r}'(t)|^3}κ(t)=∣r′(t)∣3∣r′(t)×r′′(t)∣​

when the denominator is nonzero.

Key results​

Vector differentiation is componentwise:

ddt⟨f(t),g(t),h(t)⟩=⟨f′(t),g′(t),h′(t)⟩.\frac{d}{dt}\langle f(t),g(t),h(t)\rangle = \langle f'(t),g'(t),h'(t)\rangle.dtd​⟨f(t),g(t),h(t)⟩=⟨f′(t),g′(t),h′(t)⟩.

Dot and cross products obey product rules:

ddt[u⋅v]=u′⋅v+u⋅v′,\frac{d}{dt}[\mathbf{u}\cdot\mathbf{v}] = \mathbf{u}'\cdot\mathbf{v}+\mathbf{u}\cdot\mathbf{v}',dtd​[u⋅v]=u′⋅v+u⋅v′,

and

ddt[u×v]=u′×v+u×v′.\frac{d}{dt}[\mathbf{u}\times\mathbf{v}] = \mathbf{u}'\times\mathbf{v}+\mathbf{u}\times\mathbf{v}'.dtd​[u×v]=u′×v+u×v′.

If speed is constant, then velocity and acceleration are orthogonal. This follows by differentiating

v⋅v=∣v∣2.\mathbf{v}\cdot\mathbf{v}=|\mathbf{v}|^2.v⋅v=∣v∣2.

If ∣v∣\vert \mathbf{v}\vert ∣v∣ is constant, the derivative of the left side is

2v⋅a=0,2\mathbf{v}\cdot\mathbf{a}=0,2v⋅a=0,

so v⋅a=0\mathbf{v}\cdot\mathbf{a}=0v⋅a=0.

Acceleration splits into tangential and normal components:

a=aTT+aNN.\mathbf{a}=a_T\mathbf{T}+a_N\mathbf{N}.a=aT​T+aN​N.

The tangential component changes speed:

aT=ddt∣v∣.a_T=\frac{d}{dt}|\mathbf{v}|.aT​=dtd​∣v∣.

The normal component changes direction. One formula is

aN=∣a∣2−aT2.a_N=\sqrt{|\mathbf{a}|^2-a_T^2}.aN​=∣a∣2−aT2​​.

Projectile motion without air resistance has constant acceleration

a(t)=⟨0,−g⟩\mathbf{a}(t)=\langle0,-g\ranglea(t)=⟨0,−g⟩

in the plane. Integrating gives velocity and position, with constants determined by initial velocity and initial position.

A curve is regular at ttt if r′(t)≠0\mathbf{r}'(t)\ne \mathbf{0}r′(t)=0. Regularity matters because tangent direction and curvature require a nonzero velocity vector. A parametrization may stop momentarily even when the geometric curve continues smoothly, so conclusions about tangents should distinguish the parametrization from the curve itself.

Arc length can be used to reparametrize a curve. Define

s(t)=∫at∣r′(u)∣ du.s(t)=\int_a^t |\mathbf{r}'(u)|\,du.s(t)=∫at​∣r′(u)∣du.

If sss can be inverted, the curve can be written in terms of arc length. In an arc-length parametrization, speed is 111, and curvature simplifies to

κ=∣dTds∣.\kappa=\left|\frac{d\mathbf{T}}{ds}\right|.κ=​dsdT​​.

This is conceptually important even when the inversion is algebraically difficult.

The unit normal vector N\mathbf{N}N points in the direction the curve is turning:

N=T′(t)∣T′(t)∣\mathbf{N}=\frac{\mathbf{T}'(t)}{|\mathbf{T}'(t)|}N=∣T′(t)∣T′(t)​

when T′(t)≠0\mathbf{T}'(t)\ne \mathbf{0}T′(t)=0. The binormal vector is

B=T×N.\mathbf{B}=\mathbf{T}\times\mathbf{N}.B=T×N.

Together, T\mathbf{T}T, N\mathbf{N}N, and B\mathbf{B}B form the moving frame used to describe space curves.

Tangential and normal acceleration separate speed change from direction change. If an object moves at constant speed around a circle, its tangential acceleration is zero but its normal acceleration is not. This is the vector reason uniform circular motion still has acceleration.

Vector integrals are componentwise. If

v(t)=⟨f(t),g(t),h(t)⟩,\mathbf{v}(t)=\langle f(t),g(t),h(t)\rangle,v(t)=⟨f(t),g(t),h(t)⟩,

then

∫v(t) dt=⟨∫f(t) dt,∫g(t) dt,∫h(t) dt⟩.\int \mathbf{v}(t)\,dt = \left\langle\int f(t)\,dt,\int g(t)\,dt,\int h(t)\,dt\right\rangle.∫v(t)dt=⟨∫f(t)dt,∫g(t)dt,∫h(t)dt⟩.

Constants of integration become constant vectors, fixed by initial position.

A smooth curve can be studied as geometry or as motion. The same set of points may have different parametrizations with different speeds. For example, ⟨cos⁡t,sin⁡t⟩\langle\cos t,\sin t\rangle⟨cost,sint⟩ and ⟨cos⁡2t,sin⁡2t⟩\langle\cos 2t,\sin 2t\rangle⟨cos2t,sin2t⟩ trace the same unit circle, but the second moves twice as fast. Curvature is geometric and does not depend on speed, while velocity and acceleration do depend on the parametrization.

For plane curves, curvature can also be written in terms of coordinate functions:

κ(t)=∣x′(t)y′′(t)−y′(t)x′′(t)∣((x′(t))2+(y′(t))2)3/2.\kappa(t)= \frac{|x'(t)y''(t)-y'(t)x''(t)|} {\left((x'(t))^2+(y'(t))^2\right)^{3/2}}.κ(t)=((x′(t))2+(y′(t))2)3/2∣x′(t)y′′(t)−y′(t)x′′(t)∣​.

This is the two-dimensional version of the cross-product formula.

The tangent line to a vector curve at t=t0t=t_0t=t0​ is

L(s)=r(t0)+sr′(t0),\mathbf{L}(s)=\mathbf{r}(t_0)+s\mathbf{r}'(t_0),L(s)=r(t0​)+sr′(t0​),

provided r′(t0)≠0\mathbf{r}'(t_0)\ne \mathbf{0}r′(t0​)=0. The normal plane in space is

r′(t0)⋅(r−r(t0))=0.\mathbf{r}'(t_0)\cdot(\mathbf{r}-\mathbf{r}(t_0))=0.r′(t0​)⋅(r−r(t0​))=0.

These formulas mirror line and plane geometry: the tangent line uses the velocity as direction, and the normal plane uses velocity as normal.

Energy intuition can also help. Acceleration parallel to velocity changes speed; acceleration perpendicular to velocity changes direction. The dot product v⋅a\mathbf{v}\cdot\mathbf{a}v⋅a detects the tangential part. If it is positive, speed is increasing; if negative, speed is decreasing; if zero, speed is momentarily constant.

Line integrals later use the same parametrized motion language. A vector field F\mathbf{F}F evaluated along a path becomes F(r(t))\mathbf{F}(\mathbf{r}(t))F(r(t)), and the differential displacement is dr=r′(t) dtd\mathbf{r}=\mathbf{r}'(t)\,dtdr=r′(t)dt. Thus the work integral

∫CF⋅dr\int_C \mathbf{F}\cdot d\mathbf{r}∫C​F⋅dr

is built from velocity along a vector-parametrized curve.

For numerical work, vector functions are sampled componentwise. A small time step estimates displacement by

Δr≈v(t)Δt.\Delta\mathbf{r}\approx \mathbf{v}(t)\Delta t.Δr≈v(t)Δt.

More accurate methods use acceleration and higher derivatives, but the geometric interpretation remains: velocity gives the local tangent displacement.

When acceleration is constant, the vector kinematic formula is

r(t)=r0+v0t+12at2.\mathbf{r}(t)=\mathbf{r}_0+\mathbf{v}_0t+\frac12\mathbf{a}t^2.r(t)=r0​+v0​t+21​at2.

This is just the componentwise antiderivative of constant acceleration. It works in two or three dimensions as long as the acceleration vector is constant. Projectile motion is the standard example, with gravity contributing only to the vertical component in the simplest model.

Speed controls arc length but not position alone. Two particles can pass through the same point at the same time with different velocities, or trace the same curve with different timing. The vector function includes both the geometric path and the schedule along that path, which is why the parameter interval and initial conditions are part of the model.

Worked example 1: helix motion​

Problem. Let

r(t)=⟨cos⁡t,sin⁡t,t⟩.\mathbf{r}(t)=\langle \cos t,\sin t,t\rangle.r(t)=⟨cost,sint,t⟩.

Find velocity, speed, acceleration, and curvature.

Method.

  1. Differentiate position:
v(t)=r′(t)=⟨−sin⁡t,cos⁡t,1⟩.\mathbf{v}(t)=\mathbf{r}'(t)=\langle-\sin t,\cos t,1\rangle.v(t)=r′(t)=⟨−sint,cost,1⟩.
  1. Compute speed:
∣v(t)∣=sin⁡2t+cos⁡2t+1=2.|\mathbf{v}(t)| =\sqrt{\sin^2 t+\cos^2 t+1} =\sqrt{2}.∣v(t)∣=sin2t+cos2t+1​=2​.
  1. Differentiate velocity:
a(t)=r′′(t)=⟨−cos⁡t,−sin⁡t,0⟩.\mathbf{a}(t)=\mathbf{r}''(t)=\langle-\cos t,-\sin t,0\rangle.a(t)=r′′(t)=⟨−cost,−sint,0⟩.
  1. Compute the cross product:
r′(t)×r′′(t)=⟨sin⁡t,−cos⁡t,1⟩.\mathbf{r}'(t)\times\mathbf{r}''(t) =\langle \sin t,-\cos t,1\rangle.r′(t)×r′′(t)=⟨sint,−cost,1⟩.
  1. Its magnitude is
sin⁡2t+cos⁡2t+1=2.\sqrt{\sin^2 t+\cos^2 t+1}=\sqrt{2}.sin2t+cos2t+1​=2​.
  1. Curvature is
κ(t)=2(2)3=12.\kappa(t)=\frac{\sqrt2}{(\sqrt2)^3}=\frac12.κ(t)=(2​)32​​=21​.

Checked answer. The helix has constant speed 2\sqrt22​ and constant curvature 1/21/21/2. Constant speed also implies v⋅a=0\mathbf{v}\cdot\mathbf{a}=0v⋅a=0, which can be checked directly.

Indeed,

v⋅a=(−sin⁡t)(−cos⁡t)+(cos⁡t)(−sin⁡t)+1⋅0=0.\mathbf{v}\cdot\mathbf{a} =(-\sin t)(-\cos t)+(\cos t)(-\sin t)+1\cdot 0 =0.v⋅a=(−sint)(−cost)+(cost)(−sint)+1⋅0=0.

The acceleration is not zero; it points inward toward the axis of the helix's circular projection. This illustrates the difference between changing direction and changing speed.

Worked example 2: projectile motion​

Problem. A projectile starts at the origin with initial velocity

v(0)=⟨30,40⟩\mathbf{v}(0)=\langle 30,40\ranglev(0)=⟨30,40⟩

ft/s and acceleration

and the projectile returns to ground level at t=2.5t=2.5t=2.5 seconds.

The horizontal range is found by substituting t=2.5t=2.5t=2.5 into x(t)=30tx(t)=30tx(t)=30t:

x(2.5)=75.x(2.5)=75.x(2.5)=75.

So the projectile lands 757575 ft from the starting point. The vertical velocity at impact is

40−32(2.5)=−40,40-32(2.5)=-40,40−32(2.5)=−40,

which has the same magnitude as the initial vertical velocity but opposite sign because the launch and landing heights match.

The speed at impact is therefore

302+(−40)2=50 ft/s.\sqrt{30^2+(-40)^2}=50\text{ ft/s}.302+(−40)2​=50 ft/s.

This equals the initial speed 302+402=50\sqrt{30^2+40^2}=50302+402​=50 ft/s in the ideal no-air-resistance model, another consistency check.

If launch and landing heights differed, that equality would generally fail because gravity would change kinetic energy relative to the starting height.

Code​

from math import sin, cos, sqrt

def helix(t):
    r = (cos(t), sin(t), t)
    v = (-sin(t), cos(t), 1)
    a = (-cos(t), -sin(t), 0)
    speed = sqrt(sum(component * component for component in v))
    return r, v, a, speed

for t in [0, 1, 2]:
    print(t, helix(t))

Common pitfalls​

  • Confusing velocity with speed. Velocity is a vector; speed is its magnitude.
  • Differentiating vector magnitudes as if ∣r∣′=∣r′∣\vert \mathbf{r}\vert '=\vert \mathbf{r}'\vert ∣r∣′=∣r′∣. This is not generally true.
  • Forgetting that constant speed does not mean zero acceleration. Direction can change.
  • Using scalar projectile formulas without tracking vector components and units.
  • Computing curvature when r′(t)=0\mathbf{r}'(t)=\mathbf{0}r′(t)=0 without checking regularity.
  • Treating parameter ttt as arc length when speed is not 111.
