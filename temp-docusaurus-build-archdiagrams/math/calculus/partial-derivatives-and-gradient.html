Partial Derivatives and the Gradient

Partial derivatives extend the derivative to functions of several variables by changing one input at a time. The gradient collects those first partial derivatives into a vector. Together they describe slopes, tangent planes, directional derivatives, level curves, and local linear approximation for surfaces.

The main conceptual shift is that a multivariable function can change in many directions from the same point. A single derivative number is no longer enough. The gradient gives the direction of steepest increase and organizes all directional rates into one vector formula.

Definitions​

For a function f(x,y)f(x,y)f(x,y), the partial derivative with respect to xxx is

fx(x,y)=lim⁡h→0f(x+h,y)−f(x,y)h,f_x(x,y)=\lim_{h\to 0}\frac{f(x+h,y)-f(x,y)}{h},fx​(x,y)=h→0lim​hf(x+h,y)−f(x,y)​,

and the partial derivative with respect to yyy is

fy(x,y)=lim⁡h→0f(x,y+h)−f(x,y)h.f_y(x,y)=\lim_{h\to 0}\frac{f(x,y+h)-f(x,y)}{h}.fy​(x,y)=h→0lim​hf(x,y+h)−f(x,y)​.

When computing fxf_xfx​, treat yyy as constant. When computing fyf_yfy​, treat xxx as constant.

The gradient is

∇f(x,y)=⟨fx(x,y),fy(x,y)⟩.\nabla f(x,y)=\langle f_x(x,y),f_y(x,y)\rangle.∇f(x,y)=⟨fx​(x,y),fy​(x,y)⟩.

For f(x,y,z)f(x,y,z)f(x,y,z),

∇f=⟨fx,fy,fz⟩.\nabla f=\langle f_x,f_y,f_z\rangle.∇f=⟨fx​,fy​,fz​⟩.

The directional derivative of fff at a point in the unit direction u\mathbf{u}u is

Duf=∇f⋅u.D_{\mathbf{u}}f=\nabla f\cdot\mathbf{u}.Du​f=∇f⋅u.

A level curve of f(x,y)f(x,y)f(x,y) is a curve

f(x,y)=c.f(x,y)=c.f(x,y)=c.

A level surface of f(x,y,z)f(x,y,z)f(x,y,z) is

f(x,y,z)=c.f(x,y,z)=c.f(x,y,z)=c.

Key results​

The gradient points in the direction of steepest increase. Since

Duf=∇f⋅u=∣∇f∣∣u∣cos⁡θ,D_{\mathbf{u}}f=\nabla f\cdot\mathbf{u} =|\nabla f||\mathbf{u}|\cos\theta,Du​f=∇f⋅u=∣∇f∣∣u∣cosθ,

and ∣u∣=1\vert \mathbf{u}\vert =1∣u∣=1, the maximum directional derivative is ∣∇f∣\vert \nabla f\vert ∣∇f∣, achieved when u\mathbf{u}u points in the gradient direction.

The gradient is perpendicular to level curves and level surfaces. If a path r(t)\mathbf{r}(t)r(t) stays on a level curve f(x,y)=cf(x,y)=cf(x,y)=c, then

f(r(t))=c.f(\mathbf{r}(t))=c.f(r(t))=c.

Differentiating gives

∇f(r(t))⋅r′(t)=0.\nabla f(\mathbf{r}(t))\cdot\mathbf{r}'(t)=0.∇f(r(t))⋅r′(t)=0.

Thus the gradient is orthogonal to the tangent direction r′(t)\mathbf{r}'(t)r′(t).

The tangent plane to z=f(x,y)z=f(x,y)z=f(x,y) at (a,b,f(a,b))(a,b,f(a,b))(a,b,f(a,b)) is

z=f(a,b)+fx(a,b)(x−a)+fy(a,b)(y−b).z=f(a,b)+f_x(a,b)(x-a)+f_y(a,b)(y-b).z=f(a,b)+fx​(a,b)(x−a)+fy​(a,b)(y−b).

This is the multivariable version of linearization. The corresponding linear approximation is

L(x,y)=f(a,b)+fx(a,b)(x−a)+fy(a,b)(y−b).L(x,y)=f(a,b)+f_x(a,b)(x-a)+f_y(a,b)(y-b).L(x,y)=f(a,b)+fx​(a,b)(x−a)+fy​(a,b)(y−b).

For an implicit surface F(x,y,z)=cF(x,y,z)=cF(x,y,z)=c, the tangent plane at point (a,b,c0)(a,b,c_0)(a,b,c0​) has normal vector ∇F(a,b,c0)\nabla F(a,b,c_0)∇F(a,b,c0​):

∇F(a,b,c0)⋅⟨x−a,y−b,z−c0⟩=0.\nabla F(a,b,c_0)\cdot\langle x-a,y-b,z-c_0\rangle=0.∇F(a,b,c0​)⋅⟨x−a,y−b,z−c0​⟩=0.

Second partial derivatives include fxxf_{xx}fxx​, fxyf_{xy}fxy​, fyxf_{yx}fyx​, and fyyf_{yy}fyy​. Under suitable continuity hypotheses,

fxy=fyx.f_{xy}=f_{yx}.fxy​=fyx​.

This equality is called Clairaut's Theorem.

The chain rule for a path (x(t),y(t))(x(t),y(t))(x(t),y(t)) is

ddtf(x(t),y(t))=fxdxdt+fydydt=∇f⋅r′(t).\frac{d}{dt}f(x(t),y(t)) =f_x\frac{dx}{dt}+f_y\frac{dy}{dt} =\nabla f\cdot\mathbf{r}'(t).dtd​f(x(t),y(t))=fx​dtdx​+fy​dtdy​=∇f⋅r′(t).

This is the same dot product structure that appears in directional derivatives.

Differentiability in several variables is stronger than the existence of partial derivatives. A function can have both fxf_xfx​ and fyf_yfy​ at a point and still fail to have a good tangent plane there. A common sufficient condition is that the first partial derivatives exist in a neighborhood of the point and are continuous at the point. Under that condition, the linear approximation really is the best first-order approximation.

The total differential records this approximation:

dz=fx(a,b) dx+fy(a,b) dy.dz=f_x(a,b)\,dx+f_y(a,b)\,dy.dz=fx​(a,b)dx+fy​(a,b)dy.

It estimates the output change caused by small input changes. In measurement problems, if xxx and yyy have small errors, the differential estimates the resulting error in z=f(x,y)z=f(x,y)z=f(x,y).

For functions of three variables, the gradient normal property is especially important. If F(x,y,z)=cF(x,y,z)=cF(x,y,z)=c is a level surface, then ∇F\nabla F∇F is perpendicular to the tangent plane. This is why tangent planes to spheres, ellipsoids, and implicit surfaces are often found more cleanly with gradients than by solving for one variable.

The directional derivative formula also makes angle dependence explicit. Moving perpendicular to the gradient gives zero first-order change because the dot product is zero. Moving opposite the gradient gives the steepest decrease, with value −∣∇f∣-\vert \nabla f\vert −∣∇f∣.

Second partials describe bending. The Hessian matrix

H=[fxxfxyfyxfyy]H= \begin{bmatrix} f_{xx} & f_{xy}\\ f_{yx} & f_{yy} \end{bmatrix}H=[fxx​fyx​​fxy​fyy​​]

is used in the multivariable second derivative test. Its determinant and signs determine whether the surface bends upward, downward, or like a saddle near a critical point.

Partial derivatives also have units. If T(x,y)T(x,y)T(x,y) is temperature in degrees Celsius and x,yx,yx,y are measured in meters, then TxT_xTx​ and TyT_yTy​ are degrees Celsius per meter. A directional derivative has the same units and gives the rate of temperature change per meter in the chosen direction. This unit check is useful in physical fields such as temperature, pressure, elevation, and concentration.

The gradient can be zero at flat points. When ∇f(a,b)=0\nabla f(a,b)=\mathbf{0}∇f(a,b)=0, every directional derivative is zero because ∇f⋅u=0\nabla f\cdot\mathbf{u}=0∇f⋅u=0 for every unit vector u\mathbf{u}u. This makes the point a candidate for a local extremum, but not a guarantee. A saddle point also has zero gradient, just as f(x)=x3f(x)=x^3f(x)=x3 has zero derivative at a point that is not a one-variable extremum.

For level curves on a contour map, closely spaced contours indicate a large gradient magnitude because the function changes rapidly over a short distance. Widely spaced contours indicate a small gradient magnitude. The gradient direction crosses contour lines at right angles and points toward increasing labels.

Linear approximation is local. If (x,y)(x,y)(x,y) moves too far from (a,b)(a,b)(a,b), curvature and higher-order terms can dominate. The tangent plane gives first-order behavior, and the Hessian describes the next-order correction.

Worked example 1: gradient and directional derivative​

Problem. Let

f(x,y)=x2y+3y2.f(x,y)=x^2y+3y^2.f(x,y)=x2y+3y2.

Find ∇f(2,1)\nabla f(2,1)∇f(2,1) and the directional derivative at (2,1)(2,1)(2,1) in the direction of v=⟨3,4⟩\mathbf{v}=\langle3,4\ranglev=⟨3,4⟩.

  1. Compute partial derivatives:
fx=2xy,fy=x2+6y.f_x=2xy, \qquad f_y=x^2+6y.fx​=2xy,fy​=x2+6y.
  1. Evaluate at (2,1)(2,1)(2,1):
fx(2,1)=2(2)(1)=4,fy(2,1)=22+6(1)=10.f_x(2,1)=2(2)(1)=4, \qquad f_y(2,1)=2^2+6(1)=10.fx​(2,1)=2(2)(1)=4,fy​(2,1)=22+6(1)=10.

Worked example 2: tangent plane and linear approximation​

Problem. Find the tangent plane to

z=f(x,y)=x2+y2z=f(x,y)=\sqrt{x^2+y^2}z=f(x,y)=x2+y2​

Method.

  1. Compute partial derivatives:
fx=xx2+y2,fy=yx2+y2.f_x=\frac{x}{\sqrt{x^2+y^2}}, \qquad f_y=\frac{y}{\sqrt{x^2+y^2}}.fx​=x2+y2​x​,fy​=x2+y2​y​.

  1. Write the tangent plane:

  1. For (3.1,3.9)(3.1,3.9)(3.1,3.9), the changes are

f(3.1,3.9)=3.12+3.92=9.61+15.21=24.82≈4.982.f(3.1,3.9)=\sqrt{3.1^2+3.9^2} =\sqrt{9.61+15.21} =\sqrt{24.82}\approx 4.982.f(3.1,3.9)=3.12+3.92​=9.61+15.21​=24.82​≈4.982.

The linear approximation is close because (3.1,3.9)(3.1,3.9)(3.1,3.9) is near (3,4)(3,4)(3,4). The tangent plane is not expected to remain accurate far from the base point.

The gradient at (3,4)(3,4)(3,4) is ⟨3/5,4/5⟩\langle3/5,4/5\rangle⟨3/5,4/5⟩, a unit vector pointing radially outward. This fits the geometry of f(x,y)=x2+y2f(x,y)=\sqrt{x^2+y^2}f(x,y)=x2+y2​: the steepest increase in distance from the origin is directly away from the origin.

In applications, the same computation estimates sensitivity. If xxx and yyy are measured values, the coefficients 3/53/53/5 and 4/54/54/5 tell how much each small input error contributes to the first-order output error. This is the multivariable version of using dy=f′(x)dxdy=f'(x)dxdy=f′(x)dx in one variable.

When using finite differences numerically, the same hold-one-variable-fixed interpretation applies. A central difference for fxf_xfx​ changes xxx in both directions while keeping yyy fixed. Changing both variables at once estimates a directional derivative, not a partial derivative.

The notation should reflect that distinction. The symbol ∂\partial∂ signals that other independent variables are held fixed, while d/dtd/dtd/dt along a path means every coordinate depending on ttt may change. Many multivariable mistakes come from using a partial derivative when a total derivative along a curve is required in context.

Writing the dependency first prevents that error in solutions and explanations consistently enough.

It is a small notation habit with large payoff.

Code​

def f(x, y):
    return x*x*y + 3*y*y

def partial_x(f, x, y, h=1e-5):
    return (f(x + h, y) - f(x - h, y)) / (2*h)

def partial_y(f, x, y, h=1e-5):
    return (f(x, y + h) - f(x, y - h)) / (2*h)

print(partial_x(f, 2, 1), partial_y(f, 2, 1))

Common pitfalls​

  • Forgetting to use a unit vector for directional derivatives.
  • Treating partial derivatives as if all variables change at once. One variable changes; the others are held constant.
  • Assuming ∇f=0\nabla f=\mathbf{0}∇f=0 whenever a level curve has a horizontal tangent. The gradient is normal, not tangent.
  • Writing a tangent plane without evaluating derivatives at the base point.
  • Confusing a tangent plane to z=f(x,y)z=f(x,y)z=f(x,y) with a level-surface tangent plane for F(x,y,z)=cF(x,y,z)=cF(x,y,z)=c.
  • Assuming mixed partials are equal without the needed continuity conditions.
