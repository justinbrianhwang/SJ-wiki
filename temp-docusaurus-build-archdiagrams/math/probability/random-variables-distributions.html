Random Variables and Distributions

A random variable converts outcomes into numbers. Instead of listing every detailed outcome in the sample space, we often care about a numerical summary: the number of heads, the waiting time until a machine fails, the maximum of several measurements, or whether a test result is positive. Distributions describe how probability is assigned to those numerical values.

This page is the bridge between event-based probability and the distribution-centered language used in statistics, simulation, and modeling. Lane et al. use probability distributions for binomial, Poisson, normal, and sampling-distribution examples. Here we build the general vocabulary: random variables, probability mass functions, probability density functions, and cumulative distribution functions.

Definitions​

A random variable is a function

X:Ω→RX:\Omega\to \mathbb{R}X:Ω→R

that assigns a real number to each outcome. The word "variable" is traditional, but formally XXX is a function.

A random variable is discrete if its possible values are finite or countably infinite. Its probability mass function (PMF) is

pX(x)=P(X=x).p_X(x)=P(X=x).pX​(x)=P(X=x).

The PMF satisfies

pX(x)≥0,∑xpX(x)=1.p_X(x)\ge 0,\quad \sum_x p_X(x)=1.pX​(x)≥0,x∑​pX​(x)=1.

A random variable is continuous if probabilities are described by a probability density function (PDF) fX(x)f_X(x)fX​(x) such that

P(a≤X≤b)=∫abfX(x) dx.P(a\le X\le b)=\int_a^b f_X(x)\,dx.P(a≤X≤b)=∫ab​fX​(x)dx.

The PDF satisfies

fX(x)≥0,∫−∞∞fX(x) dx=1.f_X(x)\ge 0,\quad \int_{-\infty}^{\infty} f_X(x)\,dx=1.fX​(x)≥0,∫−∞∞​fX​(x)dx=1.

The cumulative distribution function (CDF) is defined for every real random variable by

FX(x)=P(X≤x).F_X(x)=P(X\le x).FX​(x)=P(X≤x).

The CDF is nondecreasing, right-continuous, and satisfies

lim⁡x→−∞FX(x)=0,lim⁡x→∞FX(x)=1.\lim_{x\to -\infty}F_X(x)=0,\quad \lim_{x\to \infty}F_X(x)=1.x→−∞lim​FX​(x)=0,x→∞lim​FX​(x)=1.

For a continuous random variable with density fXf_XfX​,

FX(x)=∫−∞xfX(t) dt,F_X(x)=\int_{-\infty}^{x} f_X(t)\,dt,FX​(x)=∫−∞x​fX​(t)dt,

and, where differentiable,

fX(x)=FX′(x).f_X(x)=F_X'(x).fX​(x)=FX′​(x).

The support of a distribution is the set of values where the PMF or PDF is positive. A parameter is a fixed constant that indexes a family of distributions, such as the success probability ppp in a Bernoulli distribution or the rate λ\lambdaλ in an exponential distribution.

Key results​

The CDF is the most universal way to describe a distribution because it works for discrete, continuous, and mixed random variables. Probability over intervals can be recovered from it:

P(a<X≤b)=FX(b)−FX(a).P(a<X\le b)=F_X(b)-F_X(a).P(a<X≤b)=FX​(b)−FX​(a).

For continuous random variables, endpoint choices usually do not matter because

P(X=a)=∫aafX(x) dx=0.P(X=a)=\int_a^a f_X(x)\,dx=0.P(X=a)=∫aa​fX​(x)dx=0.

For discrete random variables, endpoint choices matter because points can have positive probability.

If XXX is discrete with PMF pXp_XpX​, then its CDF is a step function:

FX(x)=∑t≤xpX(t).F_X(x)=\sum_{t\le x}p_X(t).FX​(x)=t≤x∑​pX​(t).

If XXX is continuous with PDF fXf_XfX​, then probabilities are areas under the curve:

P(a≤X≤b)=∫abfX(x) dx.P(a\le X\le b)=\int_a^b f_X(x)\,dx.P(a≤X≤b)=∫ab​fX​(x)dx.

Do not interpret density height as probability. A density can exceed 111 on a short interval; the area, not the height, is the probability.

A quantile is an inverse-CDF value. A number qαq_\alphaqα​ is an α\alphaα-quantile if

P(X≤qα)≥αP(X\le q_\alpha)\ge \alphaP(X≤qα​)≥α

and

P(X≥qα)≥1−α.P(X\ge q_\alpha)\ge 1-\alpha.P(X≥qα​)≥1−α.

For continuous strictly increasing CDFs, this is simply

qα=FX−1(α).q_\alpha=F_X^{-1}(\alpha).qα​=FX−1​(α).

Two random variables can have the same distribution even if they are defined on different sample spaces. For instance, the indicator of heads in a coin toss and the indicator of drawing a red card from a balanced red/black deck are different functions on different experiments, but both can be Bernoulli(1/2)(1/2)(1/2). Distributional statements describe probabilities of values, not the underlying physical mechanism.

Random variables can also be mixed. A delivery time might have a positive probability of being exactly 000 if an item is already available, plus a continuous density over positive waiting times if it must be shipped. Such variables are neither purely discrete nor purely continuous, but their CDF still works. This is one reason the CDF is more fundamental than either a PMF or a PDF.

When modeling, the support is as important as the formula. A normal distribution may approximate adult heights well near the center but still gives tiny probability to negative heights. A beta distribution is often better for proportions because its support is exactly (0,1)(0,1)(0,1). A Poisson distribution can count events but cannot model negative values or fixed upper limits. Matching support prevents many silent modeling errors.

CDFs also make inequalities precise. The event X≤xX\le xX≤x is always meaningful, but phrases such as "around xxx" or "near xxx" require an interval. In continuous models, changing from <\lt < to ≤\le≤ usually makes no difference; in discrete models, it can change the answer by a point mass. When using software, check whether a function returns P(X≤x)P(X\le x)P(X≤x), P(X<x)P(X\lt x)P(X<x), P(X≥x)P(X\ge x)P(X≥x), or P(X>x)P(X\gt x)P(X>x).

Code​

import numpy as np

# PMF for the sum of two dice.
outcomes = [(i, j) for i in range(1, 7) for j in range(1, 7)]
sums = np.array([i + j for i, j in outcomes])

values, counts = np.unique(sums, return_counts=True)
pmf = counts / counts.sum()

for value, probability in zip(values, pmf):
    print(f"P(X={value}) = {probability:.4f}")

mask = (values >= 5) & (values <= 9)
print("P(5 <= X <= 9) =", pmf[mask].sum())

# Continuous example: F(x)=x^2 on [0,1].
def cdf(x):
    x = np.asarray(x)
    return np.where(x < 0, 0, np.where(x <= 1, x**2, 1))

print("P(0.25 <= X <= 0.75) =", cdf(0.75) - cdf(0.25))

Common pitfalls​

  • Forgetting that a random variable is a function on outcomes, not the outcome itself.
  • Treating a density value fX(x)f_X(x)fX​(x) as if it were P(X=x)P(X=x)P(X=x).
  • Ignoring support. Formulas such as 2x2x2x are densities only on the interval where they are defined.
  • Assuming endpoint choices never matter. They do not matter for continuous distributions, but they do matter for discrete distributions.
  • Using a PDF when the problem gives a CDF, or differentiating a discrete CDF as if it were smooth.
  • Failing to check that probabilities or densities normalize to one.
