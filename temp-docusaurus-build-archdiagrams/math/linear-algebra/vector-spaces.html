General Vector Spaces

The abstraction from Rn\mathbb{R}^nRn to vector spaces is not a change of subject. It is a way to recognize the same algebraic behavior in polynomials, functions, matrices, sequences, and solution sets. Once the vector space axioms hold, ideas such as span, independence, basis, dimension, and linear transformation work in the same way.

This abstraction is useful because many objects that do not look like arrows still add and scale like arrows. Polynomials can be added and multiplied by constants. Continuous functions can be added and scaled. Matrices of the same size can be added and scaled. Linear algebra keeps the operations and lets the objects vary.

Definitions​

A real vector space is a set VVV with vector addition and scalar multiplication satisfying the standard closure, associativity, commutativity, identity, inverse, and distributive axioms. The elements of VVV are called vectors even when they are functions or matrices.

A subspace WWW of VVV is a nonempty subset that is itself a vector space under the operations inherited from VVV. It is enough to check:

  1. 0∈W\mathbf{0}\in W0∈W.
  2. If u,v∈W\mathbf{u},\mathbf{v}\in Wu,v∈W, then u+v∈W\mathbf{u}+\mathbf{v}\in Wu+v∈W.
  3. If c∈Rc\in\mathbb{R}c∈R and u∈W\mathbf{u}\in Wu∈W, then cu∈Wc\mathbf{u}\in Wcu∈W.

The span of S={v1,…,vk}S=\{\mathbf{v}_1,\ldots,\mathbf{v}_k\}S={v1​,…,vk​} is

span⁡(S)={c1v1+⋯+ckvk:ci∈R}.\operatorname{span}(S)= \{c_1\mathbf{v}_1+\cdots+c_k\mathbf{v}_k:c_i\in\mathbb{R}\}.span(S)={c1​v1​+⋯+ck​vk​:ci​∈R}.

A set is linearly independent if the equation

c1v1+⋯+ckvk=0c_1\mathbf{v}_1+\cdots+c_k\mathbf{v}_k=\mathbf{0}c1​v1​+⋯+ck​vk​=0

forces every ci=0c_i=0ci​=0. If there is a nontrivial solution, the set is linearly dependent.

A basis is a linearly independent spanning set. In a finite-dimensional vector space, the dimension is the number of vectors in any basis.

Important examples include Rn\mathbb{R}^nRn, the polynomial space PnP_nPn​ of polynomials of degree at most nnn, the matrix space Mm,nM_{m,n}Mm,n​, and spaces of functions such as C[a,b]C[a,b]C[a,b].

Key results​

The span of any nonempty set of vectors is a subspace. Closure follows because a sum of linear combinations is another linear combination, and a scalar multiple of a linear combination is another linear combination.

The solution set of a homogeneous linear system Ax=0A\mathbf{x}=\mathbf{0}Ax=0 is a subspace of Rn\mathbb{R}^nRn. If Au=0A\mathbf{u}=\mathbf{0}Au=0 and Av=0A\mathbf{v}=\mathbf{0}Av=0, then

A(u+v)=Au+Av=0,A(\mathbf{u}+\mathbf{v})=A\mathbf{u}+A\mathbf{v}=\mathbf{0},A(u+v)=Au+Av=0,

and

A(cu)=cAu=0.A(c\mathbf{u})=cA\mathbf{u}=\mathbf{0}.A(cu)=cAu=0.

In contrast, the solution set of a nonhomogeneous system Ax=bA\mathbf{x}=\mathbf{b}Ax=b is usually not a subspace because it generally does not contain 0\mathbf{0}0.

Every finite spanning set can be reduced to a basis by removing redundant vectors. Every finite independent set can be extended to a basis by adding vectors until it spans. These facts are the structural reason dimension is well-defined.

A subset described by a linear condition is often a subspace when the condition is homogeneous. For example, the set of polynomials ppp with p(0)=0p(0)=0p(0)=0 is a subspace because (p+q)(0)=0(p+q)(0)=0(p+q)(0)=0 and (cp)(0)=0(cp)(0)=0(cp)(0)=0. The set of polynomials with p(0)=1p(0)=1p(0)=1 is not a subspace because it does not contain the zero polynomial and is not closed under scalar multiplication.

Worked example 1: Test a subset of polynomials​

Problem: let

W={p(x)∈P3:p(1)=0}.W=\{p(x)\in P_3:p(1)=0\}.W={p(x)∈P3​:p(1)=0}.

Show that WWW is a subspace of P3P_3P3​.

Step 1: check the zero vector. The zero polynomial satisfies

0(1)=0,0(1)=0,0(1)=0,

so 0∈W0\in W0∈W.

Step 2: check closure under addition. Suppose p,q∈Wp,q\in Wp,q∈W. Then p(1)=0p(1)=0p(1)=0 and q(1)=0q(1)=0q(1)=0. Therefore

(p+q)(1)=p(1)+q(1)=0+0=0.(p+q)(1)=p(1)+q(1)=0+0=0.(p+q)(1)=p(1)+q(1)=0+0=0.

So p+q∈Wp+q\in Wp+q∈W.

Step 3: check closure under scalar multiplication. If c∈Rc\in\mathbb{R}c∈R and p∈Wp\in Wp∈W, then

(cp)(1)=c p(1)=c⋅0=0.(cp)(1)=c\,p(1)=c\cdot0=0.(cp)(1)=cp(1)=c⋅0=0.

So cp∈Wcp\in Wcp∈W.

Checked answer: WWW is a subspace of P3P_3P3​.

We can also describe it. Since p(1)=0p(1)=0p(1)=0, the factor theorem gives

p(x)=(x−1)(ax2+bx+c).p(x)=(x-1)(ax^2+bx+c).p(x)=(x−1)(ax2+bx+c).

Thus WWW is spanned by

x−1,x(x−1),x2(x−1).x-1,\qquad x(x-1),\qquad x^2(x-1).x−1,x(x−1),x2(x−1).

Worked example 2: Show a set is not a subspace​

Problem: decide whether

S={[xy]∈R2:x+y=1}S=\left\{ \begin{bmatrix}x\\y\end{bmatrix}\in\mathbb{R}^2:x+y=1 \right\}S={[xy​]∈R2:x+y=1}

is a subspace of R2\mathbb{R}^2R2.

Step 1: test the zero vector.

[00]\begin{bmatrix}0\\0\end{bmatrix}[00​]

has x+y=0x+y=0x+y=0, not 111. Therefore 0∉S\mathbf{0}\notin S0∈/S.

Step 2: since every subspace must contain the zero vector, SSS is not a subspace.

Step 3: confirm by scalar multiplication. The vector

u=[10]\mathbf{u}= \begin{bmatrix}1\\0\end{bmatrix}u=[10​]

belongs to SSS because 1+0=11+0=11+0=1. But

2u=[20]2\mathbf{u}= \begin{bmatrix}2\\0\end{bmatrix}2u=[20​]

does not belong to SSS because 2+0≠12+0\neq12+0=1.

Checked answer: SSS is not a subspace. Geometrically, it is a line that does not pass through the origin.

Code​

import sympy as sp

x = sp.symbols("x")
basis = [x - 1, x * (x - 1), x**2 * (x - 1)]

for p in basis:
    print(sp.expand(p), "p(1) =", sp.expand(p).subs(x, 1))

candidate = 2*x**3 - 3*x**2 + x
print(sp.factor(candidate))
print(candidate.subs(x, 1))

The snippet uses symbolic polynomials to check the condition p(1)=0p(1)=0p(1)=0 and to factor a candidate polynomial. It illustrates that vector-space ideas apply to algebraic objects beyond columns of numbers.

Common pitfalls​

  • Checking only that a subset is nonempty. A subspace must also be closed under addition and scalar multiplication.
  • Forgetting that the zero vector depends on the space: zero polynomial, zero matrix, zero function, or zero column vector.
  • Treating any line in R2\mathbb{R}^2R2 as a subspace. Only lines through the origin are subspaces.
  • Assuming a set described by an equation is a subspace. The equation must interact correctly with addition and scaling, and nonhomogeneous constants usually break closure.
  • Confusing span with independence. A set can span and still be dependent if it contains redundancy.
  • Believing vectors must be arrows. In a vector space, "vector" means an object that obeys the vector operations.

When testing a subspace, the zero-vector check is the fastest first filter. If the zero vector is missing, the set is not a subspace. If the zero vector is present, closure still has to be checked. Many students stop too early after finding zero, but zero is necessary rather than sufficient. Closure under linear combinations is the compact test:

u,v∈W⟹cu+dv∈W\mathbf{u},\mathbf{v}\in W \quad\Longrightarrow\quad c\mathbf{u}+d\mathbf{v}\in Wu,v∈W⟹cu+dv∈W

for all scalars c,dc,dc,d.

Homogeneous conditions tend to define subspaces because scaling preserves zero. Conditions such as p(1)=0p(1)=0p(1)=0, Ax=0A\mathbf{x}=\mathbf{0}Ax=0, or f′′+f=0f''+f=0f′′+f=0 are compatible with addition and scalar multiplication when the underlying operation is linear. Nonhomogeneous conditions such as p(1)=1p(1)=1p(1)=1 or Ax=bA\mathbf{x}=\mathbf{b}Ax=b with b≠0\mathbf{b}\neq\mathbf{0}b=0 usually define shifted sets, not subspaces.

The abstraction also clarifies why proofs can be reused. Once a theorem is proved from the vector space axioms, it applies to polynomials, matrices, functions, and coordinate vectors without separate proofs. This is the payoff for learning the axioms: they identify the exact assumptions needed for linear arguments to work.

For finite-dimensional spaces, basis and dimension prevent ambiguity. A polynomial in P3P_3P3​ can be represented by its coefficient vector relative to {1,x,x2,x3}\{1,x,x^2,x^3\}{1,x,x2,x3}, but it is still a polynomial. Coordinates are representations of vectors, not the vectors themselves. This distinction becomes important when changing bases or choosing inner products.

Examples and nonexamples are equally important. The set of all polynomials is a vector space, and the set PnP_nPn​ is a finite-dimensional subspace. The set of polynomials of exactly degree nnn is not a subspace because adding two degree-nnn polynomials can cancel leading terms and produce lower degree, and the zero polynomial does not have degree nnn. This kind of closure failure is common.

Subspaces can be combined. The intersection of two subspaces is always a subspace because any vector satisfying both sets of closure-compatible conditions remains inside both after addition and scaling. The union of two subspaces is usually not a subspace unless one subspace contains the other. This distinction is a useful test of whether a proposed construction respects linear structure.

The language of vector spaces prepares for linear maps. Kernels, ranges, eigenspaces, and solution sets of homogeneous systems are all subspaces. Once you recognize a set as a subspace, the next natural questions are: what spans it, what basis is convenient, and what is its dimension?

A vector space is not defined by what its elements look like; it is defined by what operations are allowed and what laws they satisfy. This is why the same theorem can apply to arrows, polynomials, matrices, and functions. The abstraction removes irrelevant surface features and keeps the additive and scalable structure.

When learning a new vector space, identify its zero vector, its addition rule, and its scalar multiplication rule first. Then ask which familiar linear algebra concepts make sense there: span, independence, basis, dimension, linear maps, and inner products if one is supplied.

Those checks keep the abstraction tied to concrete operations.
