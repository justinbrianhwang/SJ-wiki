Joint Distributions, Transformations, and Independence

Many probability questions involve several random variables defined on the same sample space. A joint distribution records how they behave together, not merely how each behaves separately. This distinction matters: the marginal laws of XXX and YYY do not determine whether they are independent, positively related, negatively related, or constrained by an equation such as Y=X2Y=X^2Y=X2.

MIT 18.440 introduces joint mass functions, joint densities, marginal distributions, independent random variables, and distributions of functions of random variables. These ideas are the technical foundation for convolutions, conditional densities, covariance, order statistics, and the transform methods used later in the course.

Definitions​

For discrete random variables XXX and YYY, the joint probability mass function is

pX,Y(x,y)=P(X=x, Y=y).p_{X,Y}(x,y)=P(X=x,\ Y=y).pX,Y​(x,y)=P(X=x, Y=y).

The marginal mass functions are obtained by summing:

pX(x)=∑ypX,Y(x,y),pY(y)=∑xpX,Y(x,y).p_X(x)=\sum_y p_{X,Y}(x,y), \qquad p_Y(y)=\sum_x p_{X,Y}(x,y).pX​(x)=y∑​pX,Y​(x,y),pY​(y)=x∑​pX,Y​(x,y).

For continuous random variables, a joint density fX,Yf_{X,Y}fX,Y​ satisfies

P((X,Y)∈A)=∬AfX,Y(x,y) dx dy.P((X,Y)\in A)=\iint_A f_{X,Y}(x,y)\,dx\,dy.P((X,Y)∈A)=∬A​fX,Y​(x,y)dxdy.

The marginal densities are

fX(x)=∫−∞∞fX,Y(x,y) dy,fY(y)=∫−∞∞fX,Y(x,y) dx.f_X(x)=\int_{-\infty}^{\infty} f_{X,Y}(x,y)\,dy, \qquad f_Y(y)=\int_{-\infty}^{\infty} f_{X,Y}(x,y)\,dx.fX​(x)=∫−∞∞​fX,Y​(x,y)dy,fY​(y)=∫−∞∞​fX,Y​(x,y)dx.

Random variables XXX and YYY are independent if for all suitable sets A,BA,BA,B,

P(X∈A, Y∈B)=P(X∈A)P(Y∈B).P(X\in A,\ Y\in B)=P(X\in A)P(Y\in B).P(X∈A, Y∈B)=P(X∈A)P(Y∈B).

In the discrete case, this is equivalent to

pX,Y(x,y)=pX(x)pY(y)p_{X,Y}(x,y)=p_X(x)p_Y(y)pX,Y​(x,y)=pX​(x)pY​(y)

for all x,yx,yx,y. In the continuous case, it is equivalent to

fX,Y(x,y)=fX(x)fY(y)f_{X,Y}(x,y)=f_X(x)f_Y(y)fX,Y​(x,y)=fX​(x)fY​(y)

where densities are defined.

Key results​

If Y=g(X)Y=g(X)Y=g(X) and ggg is strictly increasing, then

FY(a)=P(Y≤a)=P(g(X)≤a)=P(X≤g−1(a))=FX(g−1(a)).F_Y(a)=P(Y\le a)=P(g(X)\le a)=P(X\le g^{-1}(a))=F_X(g^{-1}(a)).FY​(a)=P(Y≤a)=P(g(X)≤a)=P(X≤g−1(a))=FX​(g−1(a)).

If ggg is differentiable with differentiable inverse, the density form is

fY(y)=fX(g−1(y))∣ddyg−1(y)∣.f_Y(y)=f_X(g^{-1}(y))\left|\frac{d}{dy}g^{-1}(y)\right|.fY​(y)=fX​(g−1(y))​dyd​g−1(y)​.

For a non-one-to-one transformation, sum the contributions from each preimage:

fY(y)=∑x:g(x)=yfX(x)∣ddyx(y)∣.f_Y(y)=\sum_{x:g(x)=y} f_X(x)\left|\frac{d}{dy}x(y)\right|.fY​(y)=x:g(x)=y∑​fX​(x)​dyd​x(y)​.

For a two-dimensional differentiable one-to-one transformation (U,V)=T(X,Y)(U,V)=T(X,Y)(U,V)=T(X,Y) with inverse (x,y)=T−1(u,v)(x,y)=T^{-1}(u,v)(x,y)=T−1(u,v),

fU,V(u,v)=fX,Y(x(u,v),y(u,v))∣det⁡∂(x,y)∂(u,v)∣.f_{U,V}(u,v) = f_{X,Y}(x(u,v),y(u,v)) \left| \det \frac{\partial(x,y)}{\partial(u,v)} \right|.fU,V​(u,v)=fX,Y​(x(u,v),y(u,v))​det∂(u,v)∂(x,y)​​.

Even when two random variables have the same marginal distribution, their joint laws can be completely different. For example, if XXX is uniform on [0,1][0,1][0,1], then Y=XY=XY=X has the same marginal as XXX but is not independent of XXX. If YYY is separately sampled uniform on [0,1][0,1][0,1], then XXX and YYY can be independent.

The marginalization formulas are probability versions of "ignore one coordinate". In a joint table, summing a row forgets which value of YYY occurred and keeps only the value of XXX. In a joint density, integrating over yyy performs the same operation continuously. This operation always loses information: many different joint laws can have the same marginals.

Independence is a factorization statement. In a finite table, every entry must equal row sum times column sum. Geometrically, the joint distribution has no interaction term; the probability assigned to a rectangle A×BA\times BA×B is the product of the two side probabilities. For continuous variables, this means the density surface separates into an xxx-part and a yyy-part.

Transformations require special care because density is tied to scale. If Y=2XY=2XY=2X, intervals in YYY correspond to intervals half as long in XXX, so the density height changes by a factor of 1/21/21/2. The derivative factor in the change-of-variables formula is exactly this scale correction. In multiple dimensions, the absolute Jacobian determinant measures how areas or volumes are distorted.

The transformation method has two complementary approaches. The CDF method asks for P(g(X)≤a)P(g(X)\le a)P(g(X)≤a) and is often best when the event has a simple inequality description. The density method uses inverse branches and derivatives and is often faster when the transformation is monotone or piecewise monotone. Both methods should give the same answer when applied correctly.

Joint distributions are also the setting for conditional densities. If XXX and YYY have joint density fX,Yf_{X,Y}fX,Y​, then the conditional density of XXX given Y=yY=yY=y is formally

fX∣Y(x∣y)=fX,Y(x,y)fY(y)f_{X\mid Y}(x\mid y)=\frac{f_{X,Y}(x,y)}{f_Y(y)}fX∣Y​(x∣y)=fY​(y)fX,Y​(x,y)​

when fY(y)>0f_Y(y)\gt 0fY​(y)>0. This formula is the continuous analogue of dividing a joint probability table entry by a column total. It prepares for conditional expectation and total variance.

The matrix picture is a useful diagnostic for independence. Once row and column sums are known, an independent joint table is forced: it must be the outer product of the marginal vectors. If the actual table differs from that outer product, the variables are dependent. For continuous variables, the same idea is harder to see visually, but a product density has rectangular probabilities that factor exactly.

For transformations, the table reminds us that discrete and continuous cases use different bookkeeping. A discrete transformation moves point masses and combines masses that land on the same value. A continuous transformation moves density through a change of scale. Forgetting this distinction leads to common errors such as assigning positive probability to a point in a continuous model or omitting a Jacobian factor.

Worked example 1: checking independence from a joint table​

Problem: Let X,YX,YX,Y take values in {0,1}\{0,1\}{0,1} with joint probabilities

Y=0Y=0Y=0Y=1Y=1Y=1
X=0X=0X=00.300.300.300.200.200.20
X=1X=1X=10.150.150.150.350.350.35

Find the marginal distributions and decide whether XXX and YYY are independent.

Method:

  1. Sum rows for XXX:
P(X=0)=0.30+0.20=0.50,P(X=0)=0.30+0.20=0.50,P(X=0)=0.30+0.20=0.50, P(X=1)=0.15+0.35=0.50.P(X=1)=0.15+0.35=0.50.P(X=1)=0.15+0.35=0.50.
  1. Sum columns for YYY:
P(Y=0)=0.30+0.15=0.45,P(Y=0)=0.30+0.15=0.45,P(Y=0)=0.30+0.15=0.45, P(Y=1)=0.20+0.35=0.55.P(Y=1)=0.20+0.35=0.55.P(Y=1)=0.20+0.35=0.55.
  1. If independent, we would need
P(X=0,Y=0)=P(X=0)P(Y=0)=0.50⋅0.45=0.225.P(X=0,Y=0)=P(X=0)P(Y=0)=0.50\cdot0.45=0.225.P(X=0,Y=0)=P(X=0)P(Y=0)=0.50⋅0.45=0.225.
  1. But the table gives
P(X=0,Y=0)=0.30.P(X=0,Y=0)=0.30.P(X=0,Y=0)=0.30.

Checked answer: XXX and YYY are not independent. The marginals alone do not reveal this; the joint table does.

Worked example 2: transforming XXX to Y=X2Y=X^2Y=X2​

Problem: Let XXX be uniform on [−1,1][-1,1][−1,1] and define Y=X2Y=X^2Y=X2. Find the density of YYY.

Method:

  1. The support of YYY is [0,1][0,1][0,1].
  2. For 0<y<10\lt y\lt 10<y<1, the equation x2=yx^2=yx2=y has two solutions:
x=y,x=−y.x=\sqrt y,\qquad x=-\sqrt y.x=y​,x=−y​.
  1. The density of XXX is fX(x)=1/2f_X(x)=1/2fX​(x)=1/2 on [−1,1][-1,1][−1,1].
  2. For the positive branch, x=yx=\sqrt yx=y​, so
∣dxdy∣=12y.\left|\frac{dx}{dy}\right|=\frac{1}{2\sqrt y}.​dydx​​=2y​1​.
  1. For the negative branch, x=−yx=-\sqrt yx=−y​, so
∣dxdy∣=12y.\left|\frac{dx}{dy}\right|=\frac{1}{2\sqrt y}.​dydx​​=2y​1​.
  1. Add both contributions:
fY(y)=12⋅12y+12⋅12y=12y,0<y<1.f_Y(y)=\frac12\cdot\frac{1}{2\sqrt y} +\frac12\cdot\frac{1}{2\sqrt y} =\frac{1}{2\sqrt y}, \qquad 0<y<1.fY​(y)=21​⋅2y​1​+21​⋅2y​1​=2y​1​,0<y<1.

Checked answer: integrate the density:

∫0112y dy=[y]01=1.\int_0^1 \frac{1}{2\sqrt y}\,dy =\left[\sqrt y\right]_0^1 =1.∫01​2y​1​dy=[y​]01​=1.

So it is a valid density.

Code​

import numpy as np

joint = np.array([[0.30, 0.20],
                  [0.15, 0.35]])

px = joint.sum(axis=1)
py = joint.sum(axis=0)
independent_table = np.outer(px, py)

print("pX:", px)
print("pY:", py)
print("independent table would be:")
print(independent_table)
print("is independent?", np.allclose(joint, independent_table))

def density_y_square(y):
    y = np.asarray(y)
    out = np.zeros_like(y, dtype=float)
    mask = (y > 0) & (y < 1)
    out[mask] = 1 / (2 * np.sqrt(y[mask]))
    return out

grid = np.linspace(0.001, 0.999, 1000)
approx_integral = np.trapz(density_y_square(grid), grid)
print("approx integral:", approx_integral)

Common pitfalls​

  • Assuming marginals determine the joint distribution. They do not.
  • Checking independence at only one point in a joint table. Independence requires factorization everywhere.
  • Forgetting the absolute derivative factor when transforming a density.
  • Using a one-to-one change-of-variables formula for a many-to-one map like Y=X2Y=X^2Y=X2.
  • Treating zero-probability conditioning events in continuous models as if the elementary discrete formula applies unchanged.
