Bernoulli, Binomial, Geometric, and Negative Binomial Laws

Repeated independent trials are the simplest laboratory for random variables. A single trial with success probability ppp gives a Bernoulli random variable. Counting successes in a fixed number of trials gives a binomial random variable. Waiting for the first success gives a geometric random variable. Waiting for the rrrth success gives a negative binomial random variable.

These distributions appear early in MIT 18.440 because they connect counting, independence, expectation, variance, and recursion. They also teach a useful modeling distinction: sometimes the number of trials is fixed and the number of successes is random; sometimes the number of successes is fixed and the waiting time is random.

Definitions​

A Bernoulli random variable with parameter p∈[0,1]p\in[0,1]p∈[0,1] takes values 111 and 000 with

P(X=1)=p,P(X=0)=1−p=q.P(X=1)=p,\qquad P(X=0)=1-p=q.P(X=1)=p,P(X=0)=1−p=q.

A binomial random variable with parameters (n,p)(n,p)(n,p) counts successes in nnn independent Bernoulli trials:

P(X=k)=(nk)pk(1−p)n−k,k=0,1,…,n.P(X=k)=\binom{n}{k}p^k(1-p)^{n-k}, \qquad k=0,1,\ldots,n.P(X=k)=(kn​)pk(1−p)n−k,k=0,1,…,n.

A geometric random variable with parameter ppp is the trial number of the first success:

P(X=k)=qk−1p,k=1,2,….P(X=k)=q^{k-1}p,\qquad k=1,2,\ldots.P(X=k)=qk−1p,k=1,2,….

This convention counts the successful trial itself. Some books use the number of failures before the first success; then the support is 0,1,2,…0,1,2,\ldots0,1,2,….

A negative binomial random variable with parameters (r,p)(r,p)(r,p) is the trial number of the rrrth success:

P(X=k)=(k−1r−1)prqk−r,k=r,r+1,….P(X=k)=\binom{k-1}{r-1}p^r q^{k-r}, \qquad k=r,r+1,\ldots.P(X=k)=(r−1k−1​)prqk−r,k=r,r+1,….

The factor (k−1r−1)\binom{k-1}{r-1}(r−1k−1​) chooses the locations of the first r−1r-1r−1 successes among the first k−1k-1k−1 trials; the kkkth trial must be a success.

Key results​

For Bernoulli XXX,

E[X]=p,Var⁡(X)=p(1−p)=pq.E[X]=p,\qquad \operatorname{Var}(X)=p(1-p)=pq.E[X]=p,Var(X)=p(1−p)=pq.

This follows because X2=XX^2=XX2=X.

For X∼Binomial⁡(n,p)X\sim\operatorname{Binomial}(n,p)X∼Binomial(n,p),

E[X]=np,Var⁡(X)=npq.E[X]=np,\qquad \operatorname{Var}(X)=npq.E[X]=np,Var(X)=npq.

Proof sketch: write X=X1+⋯+XnX=X_1+\cdots+X_nX=X1​+⋯+Xn​, where the XiX_iXi​ are independent Bernoulli(p)(p)(p) indicators. Linearity gives E[X]=npE[X]=npE[X]=np. Independence gives

Var⁡(X)=∑i=1nVar⁡(Xi)=npq.\operatorname{Var}(X)=\sum_{i=1}^n\operatorname{Var}(X_i)=npq.Var(X)=i=1∑n​Var(Xi​)=npq.

For X∼Geometric⁡(p)X\sim\operatorname{Geometric}(p)X∼Geometric(p),

E[X]=1p,Var⁡(X)=qp2.E[X]=\frac1p,\qquad \operatorname{Var}(X)=\frac{q}{p^2}.E[X]=p1​,Var(X)=p2q​.

A short expectation proof uses conditioning on the first trial. Let m=E[X]m=E[X]m=E[X]. With probability ppp, X=1X=1X=1. With probability qqq, one trial is used and the remaining waiting time has the same distribution as XXX. Thus

m=p⋅1+q(1+m)=1+qm,m=p\cdot 1+q(1+m)=1+qm,m=p⋅1+q(1+m)=1+qm,

so m=1/pm=1/pm=1/p.

For X∼NegativeBinomial⁡(r,p)X\sim\operatorname{NegativeBinomial}(r,p)X∼NegativeBinomial(r,p),

E[X]=rp,Var⁡(X)=rqp2.E[X]=\frac{r}{p}, \qquad \operatorname{Var}(X)=\frac{rq}{p^2}.E[X]=pr​,Var(X)=p2rq​.

This is because the waiting time for rrr successes is a sum of rrr independent geometric waiting times, one for each successive success.

The geometric distribution has the memoryless property:

P(X>m+n∣X>m)=P(X>n).P(X>m+n\mid X>m)=P(X>n).P(X>m+n∣X>m)=P(X>n).

In words, after mmm failures, the remaining waiting time has the same distribution as the original waiting time.

The binomial and negative binomial distributions are complementary views of the same repeated-trial process. In the binomial model, time is fixed and the number of successes is random. In the negative binomial model, the number of successes is fixed and the time required is random. Many word problems can be solved only after identifying which of these two quantities is being held fixed.

For the binomial distribution, the coefficient (nk)\binom nk(kn​) counts locations of the successes. It does not count different probabilities; every sequence with kkk successes and n−kn-kn−k failures has probability pkqn−kp^kq^{n-k}pkqn−k. The coefficient appears because there are (nk)\binom nk(kn​) such sequences. This is the direct link back to the binomial theorem:

∑k=0n(nk)pkqn−k=(p+q)n=1.\sum_{k=0}^n\binom nkp^kq^{n-k}=(p+q)^n=1.k=0∑n​(kn​)pkqn−k=(p+q)n=1.

For the geometric distribution, the tail probability is especially simple:

P(X>k)=qk.P(X>k)=q^k.P(X>k)=qk.

This says the first kkk trials all failed. The tail form is often easier than summing the PMF, and it makes memorylessness immediate:

P(X>m+n∣X>m)=qm+nqm=qn.P(X>m+n\mid X>m) =\frac{q^{m+n}}{q^m} =q^n.P(X>m+n∣X>m)=qmqm+n​=qn.

Negative binomial waiting times can be decomposed as

X=G1+⋯+Gr,X=G_1+\cdots+G_r,X=G1​+⋯+Gr​,

where the GiG_iGi​ are independent geometric waiting times for successive successes. After each success, the future sequence of trials has the same distribution as a fresh sequence. This decomposition gives the expectation and variance quickly, and it also explains why the negative binomial is a discrete analogue of the gamma distribution, which is a sum of exponential waiting times.

The parameter ppp should be tied to a clearly defined success event. A "success" might mean heads, a defective item, a goal, or a baby crying in a particular minute. Changing the success definition changes ppp and may change whether trials are plausibly independent.

Read the table by first identifying the random quantity. "How many successes occur in ten trials?" points to the binomial row because the trial count is fixed. This classification step is often more important than the algebra, because the formulas can look similar but answer different questions.

Code​

from math import comb

def binomial_pmf(n, p, k):
    return comb(n, k) * (p ** k) * ((1 - p) ** (n - k))

def negative_binomial_trial_pmf(r, p, k):
    if k < r:
        return 0.0
    q = 1 - p
    return comb(k - 1, r - 1) * (p ** r) * (q ** (k - r))

print("P(Binomial(6, .5)=4):", binomial_pmf(6, 0.5, 4))
print("P(third success on trial 10):", negative_binomial_trial_pmf(3, 0.2, 10))
print("Expected third success trial:", 3 / 0.2)

# Check that the first 1000 geometric probabilities nearly sum to 1.
p = 0.2
geom_sum = sum(((1 - p) ** (k - 1)) * p for k in range(1, 1001))
print("truncated geometric mass:", geom_sum)

Common pitfalls​

  • Mixing the two geometric conventions. Always check whether XXX counts trials until success or failures before success.
  • Treating the final success in a negative binomial problem as optional. The last trial must be a success.
  • Forgetting independence in the binomial model. The formula (nk)pkqn−k\binom nkp^kq^{n-k}(kn​)pkqn−k assumes independent repeated trials with the same ppp.
  • Using binomial when the stopping rule is "continue until success". Fixed-trial and waiting-time questions have different distributions.
  • Assuming the memoryless property applies to all waiting-time distributions. In this discrete setting it is special to the geometric law.
